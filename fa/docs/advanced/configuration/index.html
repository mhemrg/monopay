پیکربندی

اگر از چند درایور استفاده می‌کنید، می توانید یک فایل جدا بسازید و همه ی پیکربندی‌ها را در آن نگه دارید.

  • JavaScript
  • TypeScript
monopay-config.js
/** @type {import("monopay").ConfigObject} */
export const monopayConfig = {
    zarinpal: {
        merchantId: "zarinpal-merchant",
        sandbox: true,
    },
    zibal: {
        merchantId: "zibal-merchant",
        sandbox: true,
    }
}
monopay-config.ts
import { ConfigObject } from "monopay";

export const monopayConfig: ConfigObject = {
  zarinpal: {
    merchantId: "zarinpal-merchant",
    sandbox: true,
  },
  zibal: {
    merchantId: "zibal-merchant",
    sandbox: true,
  },
};

و سپس در فایل هایی که به پیکربندی نیاز دارید، آن را ایمپورت کنید.
